# Draft README

## Trigger

Project README is missing, outdated, or unclear for target users.

## Workflow

1. Define purpose and audience.
2. Document prerequisites and installation.
3. Add quickstart usage examples.
4. Include configuration and environment variables.
5. Add troubleshooting and known limitations.

## Guardrails

- Write for first-time readers and repeatable onboarding.
- Keep setup steps ordered and explicit.

## Output

- Structured README draft
- Quickstart and configuration guidance
- Troubleshooting and limitations section
